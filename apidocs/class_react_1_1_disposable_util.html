static IDisposable React.DisposableUtil.Dispose ( IDisposable  disp)
static

Disposes disp and returns NOOP. This enables code like: disp = DisposableUtil.Dispose(disp); which simplifies disconnecting a given connection reference and resetting it to NOOP.

readonly IDisposable React.DisposableUtil.NOOP = Join()
static

A disposable which no-ops on Dispose and throws an exception for all other methods. This is for the following code pattern:

IDisposable _conn = DisposableUtil.NOOP;
void Open () {
   _conn = whatever.connect(...);
}
void Dispose () {
   _conn = DisposableUtil.Dispose(_conn);
}

In that it allows Dispose to avoid a null check if it's possible for Dispose to be called with no call to Open or repeatedly.
